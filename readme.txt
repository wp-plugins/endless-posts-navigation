Endless Posts Navigation is a great plugin to loop your posts/pages with alphabetic order. It is simple to use.




== Description ==
It automatically sense the post type where it is implemented and provide the endless navigation from the same taxonomy/category. You don't need to worry about its order. It automatically manage alphabetical order for next and previous navigation.

* 	an installed and configured copy of [WordPress][]

*	FTP, SFTP or shell access to your web host

= New Installations =

Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin

4. Now go to admin menu -> settings -> Endless Posts Navigation


Method-B:

1.	Download the Endless Posts Navigation installation package and extract the files on

	your computer. 
2.	Create a new directory named `Endless Posts Navigation` in the `wp-content/plugins`

	directory of your WordPress installation. Use an FTP or SFTP client to

	upload the contents of your Endless Posts Navigation archive to the new directory

	that you just created on your web host.
3.	Log in to the WordPress Dashboard and activate the Endless Posts Navigation plugin.
4.	Once the plugin is activated, a new **Endless Posts Navigation** sub-menu will appear in your Wordpress admin -> settings menu.

= Upgrades =

To *upgrade* an existing installation of Endless Posts Navigation to the most recent release:

1.	Download the Endless Posts Navigation installation package and extract the files on

	your computer. 
2.	Upload the new PHP files to `wp-content/plugins/Endless Posts Navigation`,

	overwriting any existing Endless Posts Navigation files that are there.

	

3.	Log in to your WordPress administrative interface immediately in order

	to see whether there are any further tasks that you need to perform

	to complete the upgrade.
4.	Enjoy your newer and hotter installation of Endless Posts Navigation
